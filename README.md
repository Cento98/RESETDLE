# RESETDLE
Project for fun for mi guild on wow RESET (Pokedle style)


Hola!, gracias por descargarte mi pequeño proyecto para la guild RESET!

Recuerda que necesitas tener el jdk17 para poder ejecutarlo.

Actualmente estamos en la version 1.1.0

Notas del parche 0.1.0:

    - Primera versión jugable
    - Jugadores añadidos: Bors, Snyk, Kamichi, Compy, Shabranz

Notas del parche 1.0.0:

    - Jugadores añadidos: Gorim, Squall, Xeraxy, Choq, Orocha, Barrinas, Xque, Mitsune y Penta
    - Mejorado sistema de impresión por consola
    - Añadido de colores

Notas del parche 1.1.0:

    - Jugadores añadidos: Xhordante, Trendelacaca, Ocelotte, Jude, Lovalgo, Thidrel, Shunzo, Aloreth, Xoplete, Discotheque, Akaeba, Gato y Khazerg
    - Añadido panel de jugadores no escritos
    - Añadido color por faccion al panel de no escritos
    - Añadido quitado de nombres ya escritos
